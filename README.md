🎮 GAMING.HUB - SITE E-COMMERCE DE JEUX VIDÉO
========================================================

📖 DESCRIPTION
Gaming.HUB est une plateforme e-commerce front-end dédiée à la vente de jeux vidéo (PC, PlayStation, Xbox). 
Le site propose une interface immersive, responsive et fidèle au design Figma, avec une expérience utilisateur 
optimisée : navigation fluide, gestion de panier, authentification, filtres de recherche et processus de commande simplifié.

🌐 PAGES DU SITE (5 pages)
1. index.html       → Accueil (Hero banner, jeux phares, promotions, FAQ, footer)
2. content/jeux.html        → Catalogue complet avec filtres par catégorie & recherche dynamique
3. content/panier.html      → Récapitulatif des articles, modification des quantités, calcul du total
4. content/connexion.html   → Formulaire de connexion avec validation & stockage local
5. content/inscription.html → Formulaire d'inscription avec vérification des champs & création de compte

⚙️ FONCTIONNALITÉS PRINCIPALES
✅ Design responsive & animations CSS fluides
✅ Système de panier persistant (localStorage)
✅ Authentification simulée (connexion/inscription) avec validation RegEx
✅ filtrage par catégorie
✅ Section promotions avec badges de réduction
✅ FAQ interactive (accordéon)
✅ Notifications toast lors de l'ajout au panier
✅ Compatible mobile, tablette & desktop

💳 MOYENS DE PAIEMENT & LIVRAISON
• Carte bancaire (Visa/Mastercard)
• PayPal
• Livraison standard (5-7 jours) ou express (2-3 jours)
• Paiement à la livraison disponible selon la localisation

🛠️ TECHNOLOGIES UTILISÉES
• HTML5 / CSS3 / JavaScript (Vanilla)
• Font Awesome 6 (icônes)
• Google Fonts (Jaini Purva, Inter, Poppins)
• localStorage (panier & sessions utilisateur)
• Aucune dépendance externe ni framework

🚀 INSTALLATION & UTILISATION
1. Télécharger ou cloner le projet
2. Ouvrir le dossier dans un éditeur de code (VS Code recommandé)
3. Lancer avec Live Server ou ouvrir directement `index.html` dans un navigateur
4. Naviguer entre les pages, tester le panier, l'inscription et la recherche
5. Déployer le site en ligne via Netlify :https://6a0c73d9fb202d67265fbec7--jocular-profiterole-7c748b.netlify.app
